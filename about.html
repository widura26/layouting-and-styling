Disini aku ngerancang sebuah dummy project, entah ini disebut kategori web apa. tapi tujuan aku membuat web ini agar kedepannya aku bisa ngerti cara layouting, styling yang baik dan benar. selain itu juga supaya aku punya portofolio berupa web yang aku rancang sendiri.


untuk kedepannya mungkin aku bakal ngembangin lebih lanjut. rencananya bakal ada beberapa teknologi yang aku pakai, seperti react js, tailwind untuk segi front end. Sedangkan untuk segi Mongodb atau mysql.


Web ini hanya menggunakan html, css dan javascript. javascript aku manfaatin buat request data dari file json yang aku buat sendiri. datanya ada di halaman home. disitu ada list card data yang ngambil dari file json itu mulai dari nama, judul dan tanggal. Selain itu, javascript aku manfaatin buat button collapse header. jadi kalo kalian buka web ini di hp, coba klik nama navbar 'MyBlog.com'. jadi nanti bakal muncul menunya.
